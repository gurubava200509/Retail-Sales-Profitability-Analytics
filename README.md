# E-Commerce Sales & Order Performance Dashboard

End-to-end data analytics project focused on analyzing e-commerce sales performance, customer behavior, and delivery operations using Python, MySQL, and Power BI.

## Project Overview
Built an interactive analytics dashboard using the Olist e-commerce dataset to analyze sales trends, delivery performance, customer purchasing behavior, and operational KPIs through data cleaning, SQL analysis, and visualization.

## Tools & Technologies
- Python (Pandas, NumPy)
- MySQL
- Power BI
- SQL
- Excel
- DAX

## Key Features
- Data cleaning and preprocessing using Python
- SQL-based business analysis
- KPI tracking and reporting
- Interactive Power BI dashboard
- Sales and customer trend analysis
- Delivery performance monitoring

## KPIs Analyzed
- Total Revenue
- Total Orders
- Average Delivery Days
- Late Delivery Percentage
- Order Status Distribution

## Dashboard Insights
- Top-performing states by revenue
- Customer payment preferences
- Delivery delay analysis
- Monthly sales and order trends

## Business Outcome
Transformed raw e-commerce data into actionable business insights through data analysis, KPI reporting, and interactive dashboard visualization.
